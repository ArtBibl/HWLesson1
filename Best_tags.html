Best tags.

Inline tags:

  • a - Creates a link to a URL: a web page, a section within a page, an email address...
    Also called the anchor element, where the a comes from.
  • b - Makes an element bold.
  • u - Makes an element's text underlined.
  • abbr - Defines an abbreviation, and usually includes its full description.
  • button - Defines a clickable button.
  • del - Defines text that has been deleted.
  • label - Defines a label for a form control.
  • mark - Defines highlighted text.
  • output - Defines the result of a calculation or of user action.
  • span - Defines a generic inline container of content, that does not carry any semantic value.
  • strong - Defines strong importance on text.
  • sub - Defines text that should be displayed lower.
  • time - Defines a time on a 24h clock.

Block tags:

  • p - Defines a simple paragraph of text.
  • address - Defines a block for contact information.
  • article - Defines a self-contained block of content that can exist in any context.
    It can have its own header, footer, sections... Useful for a list of blog posts.
  • canvas - Defines an element where you can draw graphics.
  • body - The container for a web page's content. Must be a direct child of 'html',
    and must be an ancestor of all HTML elements (except where noted).
  • div - Defines a generic block of content, that does not carry any semantic value.
    Used for layout elements like containers.
  • footer - Defines the footer of a web page or section.
  • h1..h5 - Defines a section heading of level one: the highest level.
  • head - Defines a container for a web page's metadata.
  • header - Defines the header of a web page or section.
  • html - Defines the root element of an HTML document.
    All other elements must be contained within this root element.
  • nav - Defines a section with navigation links.
  • section - Defines a section within a web page.
